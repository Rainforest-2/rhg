# Phase 1 CustomStage schema v3 fixtures

Deterministic, non-personal fixtures used by `check-custom-stage-schema-v3.mjs` and the
CustomStage foundation test. They demonstrate the v2→v3 null migration, a canonical v3
restriction, the export envelope, and the separate provenance localStorage key. No browser
state, credential, cookie, token, or user-provided content is included.
